Javascript Objects

There are two ways to construct a javascript object. Literal notation and constructor notation. Literal notation looks like this:

var snoopy = { species: "beagle", age: 10 };

Constructor notation looks like this:

var buddy = new Object(); buddy.species = "golden retriever"; buddy.age = 5;

Creating and calling a function:

var multiply = function (x, y) { return x * y; }; multiply(2, 3);

A function inside an object is called a method:

var bob = new Object(); bob.name = "Bob Smith"; bob.age = 30; // this time we have added a method, setAge bob.setAge = function (newAge){ bob.age = newAge; };

To call a method first type the object then dot notation for method:

bob.setAge(40);

Methods can make calculation with the object's properties not only parameters given.

var bob = new Object(); bob.age = 17; // this time we have added a method, setAge bob.setAge = function (newAge){ bob.age = newAge; }; bob.getYearOfBirth = function () { return 2014 - bob.age; };

You can create a method that works for many objects by using the keyword "this":

var setAge = function (newAge) { this.age = newAge; }; // now we make bob var bob = new Object(); bob.age = 30; bob.setAge = setAge;

You have to set the method equal to itself before calling it:

var susan = new Object(); susan.age = 25; susan.setAge = setAge; susan.setAge(35);

A method that is created with "this" but also with a specific object name will not have to be set:

var rectangle = new Object(); rectangle.height = 3; rectangle.width = 4; // here is our method to set the height rectangle.setHeight = function (newHeight) { this.height = newHeight; };

You can make your own object constructors:

function Person(name,age) { this.name = name; this.age = age; }

FOR LOOPS

for ( initialization; condition; increment) { // code to run each iteration }

var i; for (i = 0; i < 2; i++) { console.log("i is now equal to " + i); }

WHILE LOOP

var i = 0; while ( i < 2 ) { console.log( "i is now " + i ); i++; }
