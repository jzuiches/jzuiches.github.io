These are some of the more popular built-in methods available to ruby arrays. Arrays are important and are less intuitive than other types like strings, integers, or hashes. Some methods for those types of objects are below the top ten.

  4. #collect
  5. #length
  6. #inject [1, 2, 3, 4].inject(0) { |result, element| result + element } # => 10 result is the argument and element is each element. So the first iteration is result(arg) + element[0] = 1. Then result will be added to element[1] and so on for a sum. In tech speak the return value of the block will be yeilded as the result argument the next time the block is executed. The inject method does not need the argument the default argument will be the first element. You can also make a hash with an array of arrays using #inject

    hash = [[:first_name, 'Shane'], [:last_name, 'Harvie']].inject({}) do |result, element| result[element.first] = element.last result end hash # => {:first_name=>"Shane", :last_name=>"Harvie"}

    Here is a nice use of inject to select and modify from an original array.array = [1, 2, 3, 4, 5, 6].inject([]) do |result, element| result << element.to_s if element % 2 == 0 result end array # => ["2", "4", "6"]
